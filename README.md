# マイン崩し

マインクラフト風ブロック崩しゲーム。回転するネザライトピッケルでブロックを採掘（破壊）する。

## 遊び方

**▶ https://kazuoharada.github.io/mine-kuzushi/**

またはローカルで `index.html` をブラウザで開くだけ（依存なしの単一ファイル）。

## 操作

| 操作 | 内容 |
|------|------|
| マウス / スライド / ←→キー | ピッケル台（パドル）を移動 |
| クリック / タップ / スペース | ゲーム開始・ピッケル発射 |
| 右上 ⏸ ボタン / P | 一時停止・再開（一時停止中は画面タップでも再開） |
| 右上 🔊 ボタン / M | ミュート切替 |

スマホ対応: 画面スライドで移動、タップで発射。アプリを切り替えたりタブが隠れると自動で一時停止する。

## ルール

- ブロックには硬さ（HP）があり、硬いブロックほど何回も掘る必要がある（ヒビが入る）
- 土(10点)・木材(15点) → 1回 / 石(25点)・石炭(35点)・鉄(50点)・金(70点) → 2回 / ダイヤ(120点) → 3回 / 黒曜石(200点) → 4回
- **TNT**: 当てると周囲8ブロックを巻き込んで爆発（TNT同士は連鎖する）
- **岩盤**: 破壊不可（レベル3以降に出現）
- ピッケルを下の溶岩に落とすとライフ（ハート）が減る。3回落とすとゲームオーバー
- 破壊可能ブロックを全部壊すとレベルクリア。レベルが上がるとボールが速くなり、台が狭くなり、硬いブロックが増える
- ハイスコアはブラウザに保存される
